# Tic-Tac-Toe Plus

Un juego de Tic-Tac-Toe (Tres en raya) desarrollado con React y Bootstrap.

## 📋 Descripción

Tic-Tac-Toe Plus es una versión mejorada del clásico juego de Tres en raya. Permite a dos jugadores competir en un tablero de 3x3, con la característica especial de poder ver el historial de movimientos y volver a cualquier jugada anterior.

## ✨ Características

- Tablero de 3x3 interactivo
- Detección automática de ganador
- Detección de empate
- Último movimiento resaltado en color verde
- Historial completo de movimientos
- Posibilidad de volver a cualquier jugada anterior
- Diseño responsive con Bootstrap
- Fondo con gradiente azul

## 🛠️ Tecnologías usadas

- React 18
- Bootstrap 5.3.3
- CSS personalizado
- react-scripts

## 🚀 Cómo correr el proyecto

1. Clona el repositorio
2. Entra a la carpeta del proyecto
3. Instala las dependencias:
```
npm install --legacy-peer-deps
```
4. Inicia el proyecto:
```
npm start
```
5. Abre tu navegador en `http://localhost:3000`

## 🎮 Cómo jugar

- El jugador **X** siempre empieza primero
- Haz clic en cualquier casilla para colocar tu ficha
- El jugador que logre poner 3 fichas en línea gana
- Puedes usar el historial de la derecha para volver a cualquier jugada anterior
- Presiona **"Iniciar partida"** para reiniciar el juego
